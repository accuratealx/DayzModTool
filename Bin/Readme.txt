Dayz Mod Tool v0.7  accuratealx@gmail


Первый запуск:
  1. Установить из Steam "Dayz Tools" и запустить один раз (запишутся ключи в реестр).
  2. Запустить один раз DayZ (создадутся недостающие каталоги).
  3. Запустить утилиту.
  4. В главном меню выбрать пункт "Закладки" -> "Запуск программ" -> "Найти приложения".
     Произойдет заполнение полных путей к приложениям.
  5. В главном меню выбрать пункт "Закладки" -> "Каталоги" -> "Добавить стандартные каталоги".
     Произойдет добавление стандартных каталогов DayZ.
  6. Пользоваться.

Особенности:
  * Настройки программы хранятся в каталоге с программой.

Настройка внешнего вида:
  * Иконки программ. В каталоге [APP]\Tabs\Launch можно заменить иконки на свои, по аналогии.
  * Можно добавить свою программу, но пока руками. Для этого добавить в List.ini новую секцию и
    положить нужные файлы иконки и параметров запуска. Сделать по аналогии свой список параметров,
    смотреть Launch.Client.Params и Launch.DedicatedServer.Params. Открывать обычным блокнотом. 
    На данный момент нет подробного описания для данного формата.
  * Кому не хватает иконок для каталогов, идем в папку "[APP]\Tabs\Directory" и копируем в нее
    недостающее. Внутри иконки должны быть два размера 16x16 и 32x32 для правильного отображения.

Ошибки:

  * Иконки флагов стран https://github.com/lipis/flag-icons.

Добавление локализации:
  1. Сделать копию файла "[APP]\Languages\English.Language"
  2. Переименовать файл на родной язык, оставив расширение
  3. Изменить значения на свои
  4. Для проверки правильности перевода запустить приложение.
